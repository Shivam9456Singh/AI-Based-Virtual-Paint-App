About App

This App works on the latest mediapipe and opencv frameworks that can detect hand motions and perform tasks in real Time. Use of Two Fingers is dine to switch to selection mode and for Switching to Drawing Mode again use index finger only

Data Collection

The data is collected by the App in the form of live cam and then transferred to the python module , where after compiling the data the real time image is converted to a grayscale and inverted Image.

Python Module

Python Module File is a full of piece of code that is working as a backend and is encoding the data and is using the necessary HandTrackingModule present int the python files to detect motion of Hands and with the collected data it is compiling our real Time Video image to Output.

Hand Tracking Module

This Module is the Base python file for this App as it uses the latest version of mediapipe and opencv-python to detect hand motions . Mediapipe is used here for Tracing out the coordinante points on the finger as it is officially provided by Google
